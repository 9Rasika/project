Overview

MyWaste is an educational web platform designed to spread awareness about household waste management and sustainable living practices. The website helps users understand different types of waste, proper disposal methods, recycling techniques, and eco-friendly habits that contribute to a cleaner environment. It also includes an interactive quiz to test users' knowledge about waste management and sustainability.

Features
🗑️ Waste Classification – Information about biodegradable, non-biodegradable, recyclable, e-waste, and hazardous waste.
🌱 Sustainable Living Guide – Practical tips for reducing waste and adopting eco-friendly habits.
❓ Interactive Quiz – Engaging quiz to assess and improve waste management awareness.
📞 Contact Section – Allows users to connect and share feedback or suggestions.
🎨 Responsive Design – User-friendly interface accessible across devices.
